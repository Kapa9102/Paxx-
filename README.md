# Paxx paint app developed on raylib
![](paxx.webp)
### Controls: 
   LEFT CLICK       : draw, select toolbar items \\
   HOLD RIGHT CLICK : drag\\
   ESCAPE           : quit\\
   D                : darkmode\\

You only need raylib to build this. 

### Building: 

gcc build.c -o build && ./build && ./main
